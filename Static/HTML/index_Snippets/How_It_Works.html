How It Works

Once an organization has a Client Account, they can begin managing members, Smart TVs and Content. This includes adding members, registering devices, and planning how they want to distribute content.
Members of the organization can generate their own content and send it to a desired Smart TV. An end-user, say, a passer-by in the hallway, will be able to view this content on that Smart TV.
The end user will be able to interact with the set of mutable content on the Smart TV by reading content and scanning the QR codes with their smartphones relative to the content they were posted with to learn more.
There will also be a unique QR code per TV - this code will allow the user to navigate to the InfoKiosk website. From here, they will be able to request their own content be pushed to a TV.
