The Cloud

Another central part of the requirements were that we had to utilize the cloud to do our processing. This involved hosting all of our computations, data and assets . One of our earliest design decisions was to use Amazon Web Services as our cloud platform. This was because BizCloud utilized AWS, but also because they offered RDS hosting, an IOT platform to handle our smart device elements, MQTT asynchronous messaging, S3 cloud storage, and the ability to deploy Lambda code segments.
